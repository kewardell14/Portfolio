Design a new home page and personal dashboard to be less intimidating and more task-oriented instead of advertising-oriented. They would like to stand out from their competitors by making their site more user-oriented.

Client:

Chase Banking

Disciplines:

User Research, Task Flow, Typography, and Prototypes

This project, in addition to concentrating on creating a user-focused design for both a main screen and dashboard, allowed me to create a mobile version of the website that helped me focus on spacing and emphasis. I also focused on creating a dark mode that would work for both the web and mobile website.
